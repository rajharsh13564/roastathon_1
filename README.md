# Roastathon Chat Web

Welcome to the Roastathon Chat Web project! This application allows users to interact with a witty roast generator and image creator using the Gemini and NanoBanana APIs.

## Project Structure

```
roastathon-chat-web
├── public
│   └── index.html        # HTML structure for the chat interface
├── src
│   ├── css
│   │   └── styles.css    # CSS styles for the chat interface
│   └── js
│       ├── main.js       # JavaScript for handling user interactions
│       └── api.js        # API integration for generating roasts and images
├── .env.example           # Template for environment variables
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd roastathon-chat-web
   ```

2. **Install Dependencies**
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Copy the `.env.example` file to `.env` and fill in your API keys:
   ```
   VITE_GEMINI_API_KEY=your_gemini_api_key
   VITE_NANOBANANA_API_KEY=your_nanobanana_api_key
   ```

4. **Run the Application**
   Start the development server:
   ```bash
   npm run dev
   ```

5. **Open in Browser**
   Navigate to `http://localhost:3000` (or the specified port) to view the application.

## Usage Guidelines

- Enter a message in the input field and click the "Send" button to receive a roast.
- You can also request an image based on a prompt.
- Enjoy the witty comebacks and creative images generated by the application!
